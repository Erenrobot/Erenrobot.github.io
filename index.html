My Projects

Language-Based Robot Navigation (2024)

Designed and built a humanoid robot that receives multi-language human orders via S2T and performs tasks using LLMs. Integrated 2D Lidar and 3D stereo camera for autonomous navigation and environment analysis.

Detection of Dental Filling Quality (2022–2023)

Created an AI tool for detecting dental fillings using X-ray images with a 0.7 mAP50 rate in the Medipol EndoAI group.

Map Merging in Multiple Robots (2021–2022)
